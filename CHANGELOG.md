# Changelog

本项目的变更记录按照版本整理，优先记录对用户、部署和兼容性有影响的内容。

## [Unreleased]

- 暂无待发布变更

## [1.2.1] - 2026-03-30

- 修复 Docker 部署时会话认证在隔天或容器重启后失效，导致重复要求输入 API 密钥的问题
- 修复未认证状态下使用邮箱恢复后，关闭“邮箱恢复已发送”提示窗会直接进入首页、没有回到强制 API 验证页的问题
- 修复历史记录管理中种子信息卡片高度过矮，导致标题和元信息显示不完整的问题

## [1.2.0] - 2026-03-22

- 首页与快捷入口样式统一，优化 PT 站点管理卡片与调度卡片布局
- 支持按站点独立控制检查间隔、清理间隔、自动下载和自动清理
- 增加三种访问控制模式：仅局域网、仅白名单 IP、局域网和公网都允许
- 认证流程升级为主密钥 + 会话 token，并支持恢复码重置
- 日志系统和筛选分类优化，支持更稳定的认证与运行日志查看
- 新增独立 Git 仓库基础设施，以及 GitHub Actions CI 与 Release Package 工作流
- 新增发布白名单导出脚本，避免本地配置、密钥、日志和临时文件进入发布包
- 强化 `.gitignore` 与 `.dockerignore`，默认忽略配置文件变体、`.env`、密钥和运行时数据
- Docker 发布流程整理，提供 `1.2.0` 和 `latest` 镜像标签
